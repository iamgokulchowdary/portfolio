Project Overview

GC Bank is a dynamic banking simulation application crafted with Python, TKInter, and SQLite3, designed to replicate real-world banking functionalities. The application includes key features such as secure user login, account management, fund transfers, and employee login, providing a comprehensive simulation of banking operations. Its use of TKInter for the graphical user interface ensures an intuitive and interactive experience for users.

This project showcases my ability to develop secure and efficient financial software by integrating robust backend processes with a user-friendly interface. By handling various banking operations and maintaining data integrity, GC Bank demonstrates my proficiency in creating reliable and functional applications that meet industry standards.
